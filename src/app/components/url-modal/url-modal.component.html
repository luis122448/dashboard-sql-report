Power BI Example

To use this API in Power BI, you need to create two parameters:
- Url: Assign the value from the input above.
- Token: Obtain the token from the ERP TSI. 


let
    UrlParam = Text.From(Url),
    TokenParam = Text.From(Token),
    SourceBinary = Web.Contents(UrlParam, [Headers=[Authorization="Bearer " & TokenParam]]),
    ParquetData = Parquet.Document(SourceBinary)
in
    ParquetData
